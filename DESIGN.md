# Mortis — DESIGN.md

## 브랜드 정체성

**팀:** 故래소  
**제품:** Mortis  
**톤:** deadpan 개그 — 진지한 척하는데 사실 웃김. 디자인은 클린, 유머는 카피에서.

> 디자인은 장례식장처럼 단정하게. 카피는 고인의 유언처럼 무겁게 (근데 웃김).

### 포인트 컬러 선택 이유
일반 금융 앱은 파란색 → 차별화. 장례식의 황금 촛불 → `#C9A84C` 골드.  
보고서는 크림색 실제 문서처럼 → 다크 배경 위에서 임팩트 있음.

## 카피 원칙

- 일반적인 금융 앱 언어 금지: "분석 완료", "리포트 생성됨"
- 장의사 언어를 deadpan으로: "사망진단서가 발급됐습니다", "아직 살아있습니다"
- 유머는 과하지 않게 — 한 문장에 한 번

### 예시
| 상황 | 일반 | Mortis |
|------|------|--------|
| 매수 체결 | "매수 완료" | "체결됐습니다. 이제 돌아올 수 없습니다." |
| 보유 중 수익 | "수익 중" | "아직 살아있습니다 +8.2%" |
| 보유 중 손실 | "손실 중" | "위험 신호 감지" |
| 매도 후 손실 | "손실 확정" | "사망진단서가 발급됐습니다" |
| 유언장 위반 | "원칙 위반 감지" | "유언장을 어기려 하십니까?" |
| 빈 화면 | "거래 내역 없음" | "아직 아무도 죽지 않았습니다" |

## 컴포넌트

### 상태 배지
- `생존 중` — 초록 (`#22C55E`)
- `위험 신호` — 빨강 (`#EF4444`)
- `관찰 중` — 골드 (`#C9A84C`)
- `사망 확인` — 회색 (`#6B7A99`)

### 보고서 카드 (사망진단서)
- 배경: 크림색 `#F8F6F0`
- 텍스트: 다크 `#1A1A1A`
- 레이아웃: 실제 의료 문서처럼 — 항목 / 값 구조
- 보더: 1px solid `#D4C9B0`

### 유언장 집행 배너
- 배경: 골드 10% opacity
- 보더: 1px solid `#C9A84C`
- 아이콘: ⚖️ (저울)

---

## Anti-patterns

- 파란색 CTA 금지 (일반 금융 앱 느낌)
- 과도한 애니메이션 금지 — 장례식장은 조용하다
- 이모지 남발 금지 — 한 화면에 최대 1개
- 빨간색 남발 금지 — 손실에만 사용
